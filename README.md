# 📚 Estrutura de Dados – Lista Dinâmica em Java

Este projeto apresenta a implementação de uma **Lista Dinâmica** em **Java**, desenvolvida como parte dos estudos da disciplina de **Estrutura de Dados**.

O objetivo é compreender o funcionamento interno de listas dinâmicas, sem o uso de estruturas prontas da linguagem, aplicando conceitos fundamentais como **nós, ponteiros e encadeamento**.

---

## 🧠 Conceitos abordados

- Estrutura de Dados
- Lista Dinâmica
- Lista Encadeada Simples
- Nós (Node)
- Alocação dinâmica de memória
- Inserção e remoção de elementos
- Percurso sequencial da lista

##🧱 Descrição das classes
No.java
Representa o nó da lista, contendo:

- Um valor armazenado
- Uma referência para o próximo nó

Cada nó é ligado ao seguinte, formando a lista encadeada.

## ListaDinamica.java
Responsável pela estrutura da lista, contendo métodos para:

- Inserir elementos
- Remover elementos
- Percorrer e exibir a lista
- Controlar o nó inicial


## Main.java
Classe principal utilizada para:

- Testar a lista dinâmica
- Inserir e remover valores
- Visualizar o funcionamento da estrutura


##✅Características da Lista Dinâmica

- Tamanho variável
- Cresce e diminui conforme a necessidade
- Não utiliza índices fixos
- Não depende de vetores (array)


##📌 Vantagens

- Uso eficiente de memória
- Inserção e remoção facilitadas
- Estrutura flexível


## ❌Desvantagens

- Acesso sequencial aos elementos
- Maior consumo de memória devido às referências
- Implementação mais complexa que vetores

- Desenvolver lógica de programação em Java
